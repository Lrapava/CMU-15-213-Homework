# CMU-15-213-Homework
My homework for Carnegie Mellon University's Introduction to Computer Systems course. Self study based on publicly available data.

Carnegie Mellon University's 15-213 is a course by Randal Bryant and David O'Hallaron based on their book Computer Systems: A Programmer's Perspective (3rd Edition). This course in computer systems has been adapted by educational institutions world-wide, including Stanford's CS107.
