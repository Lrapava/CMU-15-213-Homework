# CMU-15-213-Homework
My homework for Carnegie Mellon University's course Introduction to Computer Systems course. Self study based on publicly available data.

Carnegie Mellon University's 15-213/18-213/15-513 is a course by Randal Bryant and David O'Hallaron based on their book Computer Systems: A Programmer's Perspective (3rd Edition). This course in computer systems has been adapted by [educational institutions world-wide](https://csapp.cs.cmu.edu/3e/adoptions.html), including Stanford's CS107.

Some publicly available resources you might find useful if you want to give it a try yourself:
- [Videotaped lectures](https://scs.hosted.panopto.com/Panopto/Pages/Sessions/List.aspx#folderID=%22b96d90ae-9871-4fae-91e2-b1627b43e25e%22)
- [Schedule and files related to each lecture](https://www.cs.cmu.edu/afs/cs/academic/class/15213-f15/www/schedule.html)
